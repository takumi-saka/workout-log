# 筋トレログ

自分用の筋トレ記録PWAです。

## 起動

```sh
cd app
python3 -m http.server 4174
```

ブラウザで以下を開きます。

```text
http://localhost:4174
```

## 実装済み

- Day1〜Day4のメニュー選択
- 前回重量を初期入力
- 目標レップを初期入力
- 重量/レップの直接入力とプラス/マイナス入力
- スキップ
- 今回だけ代替種目
- 種目メモ
- セッション評価
- ローカル保存
- 履歴表示
- Day別の最大重量合計グラフ
- 種目別の最大重量グラフ
- CSV出力
- LLM用サマリーコピー
- メニュー名/種目の簡易編集
- PWA manifest
- Service Worker

## データ保存

データはブラウザの `localStorage` に保存します。

外部サーバーには送信しません。
